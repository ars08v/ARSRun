أداة تحرير الفصول (ARS Editor)

مميزات الأداة:

  • تعمل بكفاءة عالية على جميع مواقع المانجا والمانهوا.
  • مصممة للعمل بشكل كامل على بيئة Termux.
  • أتمتة كاملة لعمليات التقطيع والمعالجة.

طريقة التشغيل:

1. تثبيت الحزم: pkg install nodejs imagemagick git

2. إعطاء الصلاحية: chmod +x tools/ars_tool.sh

3. تفعيل الأداة: ./tools/ars_tool.sh
